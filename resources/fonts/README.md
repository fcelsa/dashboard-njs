# Local Chicago font setup

To use the classic Chicago look in themes `mac1984` and `mac1990`, place your licensed font files in this folder:

- `resources/fonts/chicago.woff2`
- `resources/fonts/chicago.woff` (optional fallback)

The app already references these files via `@font-face` in `resources/css/styles.css`.

Notes:
- This repository does not include the original Chicago font files.
- Please add only font files you are legally allowed to use and distribute.
