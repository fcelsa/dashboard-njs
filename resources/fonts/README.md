# Local Chicago font setup

To use the classic Chicago look in themes `mac1984` and `mac1990`, place these files in this folder:

- `resources/fonts/chicago.woff` (used by theme `mac1984`)
- `resources/fonts/ChicagoFLF.ttf` (used by theme `mac1990`)

The app references these files via `@font-face` in `resources/css/styles.css`.

Sources used:
- `ChicagoFLF.ttf`: https://fontlibrary.org/en/font/chicagoflf
- `chicago.woff`: https://github.com/KingDuane/Chicago-Kare

Notes:
- This repository does not include these font files by default.
- Please add only font files you are legally allowed to use and distribute.
